# Arcade Persona Lab

This repo now hosts a small Vite + Three.js prototype for an open-air arcade plaza inspired by `track_arcade_big`, but without the racetrack. The scene is built around a visible third-person persona that reacts to movement, jump, and crouch input.

## Features

- A stylized plaza environment exported from Blender as `arcade_plaza_big.glb`.
- A third-person persona built from Three.js meshes, so walk, jump, and crouch states are visible in-world.
- Keyboard and mobile-friendly controls.
- A graceful fallback arena if the GLB cannot be loaded.

## Controls

- `W`, `A`, `S`, `D` or arrow keys: move the persona
- `Space`: jump
- `Ctrl`: crouch
- Click the viewport: capture mouse look on desktop
- Touch drag: rotate the camera on mobile

## Local Development

1. Install dependencies:

   ```bash
   npm install
   ```

2. Start the Vite dev server:

   ```bash
   npm run dev
   ```

3. Open the local URL shown in the terminal.

## Blender Export

The plaza environment is generated by [`arcade_plaza_big.py`](./arcade_plaza_big.py). To rebuild the GLB:

```bash
blender -b --python arcade_plaza_big.py -- ./public/arcade_plaza_big.glb
```
