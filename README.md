# Kingfisher Lab

A bird-flight playground built with [Three.js](https://threejs.org/) and [Vite](https://vitejs.dev/). Fly a kingfisher over a procedural grass landscape — start grounded in idle, take off, bank, dive, and land.

## Controls

| Key | Action |
|---|---|
| `W` / `Space` | Take off |
| `A` / `D` | Bank left / right |
| `S` (near ground) | Land |
| `Shift` | Dive |

## Getting Started

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```
